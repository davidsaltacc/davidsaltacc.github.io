Modules
Combat
Kill Aura

Attacks everything living in the players range.

Settings
Legit

Only attack entities the player has a clear line of sight to.

Criticals

Automatically lands critical hits. For some reason only works when not moving. todo for me, fix!

Auto Crystal

Automatically detonates (not places) crystals in range.

Settings
Face Crystals

Faces the crystals (server side).

No Knockback

Prevents you from getting knockback from attacks.

Settings
Strength

How strong the knockback prevention is, 0 = none, 1 = full.

Player
Reach

Gives you extended reach.

Settings
Range

The range.

Fast Break

Allows you to break blocks faster.

Hotbar Cycle

Cycles through your hotbar really fast.

Autofish

Automatically pulls back and recasts the fishing rod.

Auto Respawn

Immediatly respawn after death.

Fast Place

Removes the item use cooldown allowing you to place blocks faster.

Movement
Airjump

Allows you to jump in the air. Hold to fly. Sprint to go insane speeds (not even intended, by the way).

Autosprint

Automatically sprint instead of walk when moving.

Parkour

Automatically jumps at the ledges of blocks (todo for me: make it so it doesn't jump when the jump is only 1 block deep).

Settings
Edge Distance

How close to the edge you have be for you to jump.

Legit

Without legit enabled, you jump way further.

Stepup

Automatically walk onto blocks without jumping.

Settings
Height

How high you can step up.

Boat Fly

Fly when in a boat (also works with other vehicles).

Settings
Speed

The speed of the boat.

No Fall Damage

You take no fall damage. As in the title.

Fast Climbing

Climb ladders & etc. faster.

High Jump

You jump higher. As in the title.

Settings
Height

The height.

No Slowdown

Eating/Using items and stepping on some blocks doesn't slow you down.

Bunny Hopping

Basically spams the jump key.

Settings
When

Options: Always, Walking, Sprinting.

Minimum Velocity

The minimum velocity required for the module to make you jump, so you don't jump while youre standing still.

Legit

Without legit, you go way faster.

Safe Walk

Prevents you from falling off platforms as if you were sneaking, without you actually having to sneak,

Fly

Fly. Its in the title.

Settings
Speed

How fast you fly.

Snowwalk

Lets you walk on powder snow, as if you were wearing leather boots.

Autoswim

Automatically start swimming when in water.

Spider

Climb up walls without ladders.

Render
Extended Cam Dist

Distances your camera further away when in third person camera mode.

Settings
Distance

The camera distance.

Night Vision

Allows you to see during the night and in dark caves.

No Pumpkin

Removes the pumpkin overlay when wearing one.

Tracers

Draws a visible line to every loaded entity.

Settings
Players

If it should be rendered for players.

Hostiles

If it should be rendered for hostile entities.

Passives

If it should be rendered for passive entities.

Ambients

If it should be rendered for ambient entities.

Others

If it should be rendered for other entities.

No Fire Overlay

Lowers or completely removes the fire overlay when on fire.

Settings
Offset

How far it should be lowered. 0 = none, 0.6 = full.

No Hurt Tilt

Stops the camera from tilting when getting hurt.

ESP

Renders a visible outline around all loaded entities.

Settings
Players

If it should be rendered for players.

Hostiles

If it should be rendered for hostile entities.

Passives

If it should be rendered for passive entities.

Ambients

If it should be rendered for ambient entities.

Others

If it should be rendered for other entities.

See Barriers

Renders the normally invisible app blocks visibly.

Camera NoClip

In third person, you can move your camera into objects.

Portal GUI

Allows you to open your inventory while inside a portal.

No Blind

Doesn't blind you when you have the blindness or darkness effect.

No Levitation

Stops you from levitating when you have the levitation effect. Useful when dealing with shulkers.

See Invisibles

Renders invisible entities as ghosts, so you can see them.

Settings
Players

If it should be rendered for players.

Hostiles

If it should be rendered for hostile entities.

Passives

If it should be rendered for passive entities.

Ambients

If it should be rendered for ambient entities.

Others

If it should be rendered for other entities.

No Wobble

Stops the screen from wobbling when inside a portal or when you have the nausea effect.

Fun
Sneak Spam

Silly. Spams the sneak key.

Settings
Speed

The speed.

Rainbow GUI

Renders the black parts of the GUI in rainbow.

Derpy

Rotates your head around like wild (only visible to other players).

Secret Shaders

Mojang coded a few shaders, most of them unused, made accessible through this module.

Settings
ID

The shader id, so which shader is getting rendered.

Dinnerbonify All

Renders all mobs upside down as if they were given the nametag "Dinnerbone" or "Grumm".

Other
Disable Updates

Disables the ShadowClient's update cycle. Most modules will stop working completely. Can help with lag.

Advanced Help

Opens this help page.

Enderman Magnet

Looks at all visible enderman in the area.

Panic

Disables all modules.

Packet Log

Logs all packets sent between the server and the client.

Settings
Mode

Which packets get logged. Options: All, Received, Sent.

Filter

A filter to filter out what you want to have logged.

Filter Mode

Options: Whitelist (allows what the filter option contains), Blacklist (blocks whatever filter contains).

Menus
Setting Menu

Opens the settings menu with settings related to the ShadowClient instance.
